# Ejemplo de entrega (sin datos reales)

Este espacio es solo para mostrar la forma de la entrega.

## Cómo se vería una entrega buena

- CSV adjunto o vinculado.
- PNG de vista previa adjunto o vinculado.
- Reflexión corta y clara.
- Nombre o alias, si está permitido.
- Nada de datos privados ni archivos personales.

## Idea clave

La persona puede haber hablado con otras, pero la evidencia y la explicación deben ser propias.
